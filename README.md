# SAI PG Website

A beautiful, modern React website for SAI PG accommodation with smooth animations and client-attracting features.

## Features

- **Hero Section**: Eye-catching landing with animated gradients and statistics
- **About Section**: Information about SAI PG with feature highlights
- **Rooms Section**: Interactive room cards with pricing for boys and girls PG
- **Amenities Section**: Grid display of all provided amenities
- **Food Menu Section**: Weekly meal schedule with timing
- **Gallery Section**: Visual tour of the facility
- **Location Section**: Both boys and girls PG locations
- **Contact Section**: Inquiry form and contact information
- **Footer**: Quick links and contact info

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository or navigate to the project folder
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` folder.

## Key Design Features

- **Smooth Animations**: Using Framer Motion for entrance and hover animations
- **Gradient Effects**: Animated gradient orbs in hero section
- **Responsive Design**: Works on all device sizes
- **Interactive Elements**: Hover effects on cards, buttons, and navigation
- **Modern Typography**: Using Poppins font for clean, modern look
- **Color Scheme**: Warm coral/red primary color with complementary accents

## Customization

You can customize the following in the code:

- **Colors**: Edit CSS variables in `:root` section of `App.css`
- **Images**: Replace icon placeholders with actual images
- **Pricing**: Update room prices in the `rooms` array in `App.jsx`
- **Contact Info**: Update phone number and addresses in respective sections
